The Butler server provides a web service for querying the LSST data release
products and retrieving the associated images and data files.  It is intended
to become the primary backend for the `Butler python library
<https://github.com/lsst/daf_butler>`_ for community science use cases.

This service is in early development and currently considered experimental.
